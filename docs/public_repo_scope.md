# Public repo scope

This repository is designed for public portfolio use.

## Included

- documentation written for public review
- derived result tables
- selected analysis scripts
- representative sample `.pts` and `.pts.v3i` files
- figures created from the uploaded project spreadsheets and assets

## Intentionally omitted

- course- or supervisor-provided executable tools
- full model folders and binary artefacts
- complete internal working directory contents
- large raw annotation trees and raw private datasets

## Why this matters

The aim is to show project contribution clearly without implying authorship of provided infrastructure.
