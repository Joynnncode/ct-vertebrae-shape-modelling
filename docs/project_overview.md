# Project overview

This project focuses on vertebral landmark localisation and 3D vertebral shape modelling from CT data.

The project workflow combined:

- manual landmark annotation and annotation review
- observer-to-observer comparison
- averaged reference point generation
- statistical shape model construction
- RFRV-CLM fitting experiments
- comparison of single-stage and two-stage search strategies

The strongest public-facing evidence preserved in this repository is:

- annotation revision summaries
- result tables exported to CSV
- analysis scripts used for comparison work
- sample annotation and image-linked files
- derived figures suitable for GitHub presentation

This repository is intentionally not a full redistribution of the original academic working directory.
