# Contribution boundaries

## Contributions this repository can safely represent as mine

- project documentation and public packaging
- annotation refinement workflow description
- result-table curation and CSV export
- comparative analysis and interpretation
- derived figures created from project outputs
- selection of representative sample files

## Materials referenced but not claimed as authored by me

- provided modelling executables
- provided lab or course toolchain components
- model framework binaries and training/search utilities
- original supervisor/course instructions

## Safe interview phrasing

> I worked on vertebral CT landmark annotation, annotation-quality refinement, experiment tracking, and comparative evaluation for a 3D shape-modelling project, using a provided modelling environment.
